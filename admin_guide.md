# Manuel d'Administration - Justine Kem's 👗

Ce site est équipé d'un système de gestion de contenu (CMS) appelé **Sanity**. Il permet à votre cliente de gérer tout le contenu du site (modèles, produits, blog, formations, témoignages) sans toucher au code.

## 1. Accéder au Studio 💻

L'administration est intégrée directement à votre site.
- **Lien d'accès** : [https://www.justinekems.com/studio](https://www.justinekems.com/studio)
- **Connexion** : Elle doit se connecter avec son compte Sanity (email/password ou Google).

---

## 2. Organisation du Studio 📁

Le menu de gauche est organisé en sections claires :

### 📸 Modèles
Permet de gérer les pièces de haute couture.
- **Ajouter** : Cliquez sur le `+`.
- **Champs** : Nom, Photo principale, Galerie (plusieurs photos), Description riche, Prix, Catégorie, Disponibilité.

### 🛍️ Catalogue
Pour les produits en vente directe (Prêt-à-porter, accessoires).
- **Champs** : Nom, Images, Description courte et longue, Prix de base, Prix promotionnel, Stock, Catégorie.

### 💬 Témoignages
Gérez les avis de vos clients.
- **Note** : De 1 à 5 étoiles.
- **Visible** : Un bouton permet de masquer/afficher un témoignage sur le site.

### 📝 Blog
Publiez des articles (en Français et Anglais).
- **Titre (FR & EN)** : Saisissez les deux versions pour le multilingue.
- **Contenu** : Éditeur de texte riche pour la mise en forme.

### 🎓 Formations
Proposez vos cours et ateliers.
- **Champs** : Titre, Image, Prix, Durée, Niveau, Lien direct vers le formulaire d'inscription.

### ⚙️ Paramètres du site
Regroupe les informations globales.
- **Contenu** : Logo, Slogans (FR/EN), Email, WhatsApp, Liens réseaux sociaux (Instagram, Facebook...), et texte du bas de page (footer).

---

## 3. Gestion des Images 🖼️

J'ai ajouté un onglet **Media** en haut du Studio. Elle peut y gérer toute sa bibliothèque d'images, les organiser et les réutiliser facilement.

> [!TIP]
> **Publication** : N'oubliez pas de cliquer sur le bouton vert **"Publish"** en bas à droite pour que les changements soient visibles sur le site public.

> [!IMPORTANT]
> **Format Image** : Pour une meilleure performance, privilégiez des images en format JPG ou WebP de moins de 1Mo.
