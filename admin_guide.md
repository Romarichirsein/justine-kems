# Manuel d'Administration - Justine Kem's 👗

Ce site est équipé d'un système de gestion de contenu (CMS) appelé **Sanity**. Il permet à votre cliente de mettre à jour le catalogue et le blog sans aucune connaissance en programmation.

## 1. Accéder à l'interface de gestion (Studio) 💻

L'administration est intégrée directement à votre site.
- **Lien d'accès** : [https://justinekems.com/studio](https://justinekems.com/studio)
- **Connexion** : Connectez-vous avec l'adresse email associée au projet Sanity.

---

## 2. Ajouter un Nouveau Produit 👜

1. Dans le menu de gauche, cliquez sur **Produit**.
2. Cliquez sur le bouton **Créer un nouveau** (icône plus ↑).
3. Remplissez simplement les champs :
   - **Nom du produit** : Le nom de la création.
   - **Catégorie** : Choisissez dans la liste déroulante (Haute Couture, Prêt-à-porter, etc.).
   - **Image principale** : Téléchargez la photo de face.
   - **Galerie** : Ajoutez d'autres photos de détails si nécessaire.
   - **Prix** : Indiquez le montant en FCFA.
4. Cliquez sur **Publish** pour mettre en ligne instantanément.

---

## 3. Ajouter un Article de Blog ✍️

1. Dans le menu de gauche, cliquez sur **Article de blog**.
2. Cliquez sur **Créer un nouveau**.
3. Remplissez champs :
   - **Titre** : Le titre de l'article.
   - **Contenu** : Écrivez votre texte (possibilité de mettre en gras, ajouter des liens, etc.).
   - **Image principale** : Photo de couverture.
   - **Catégorie** : Mode, Tendances, Événements, etc.
4. Cliquez sur **Publish**.

---

## 4. Gérer les Témoignages ⭐

Elle peut également ajouter ou modifier les avis clients dans la section **Témoignages** pour qu'ils apparaissent sur la page d'accueil.

---

> [!TIP]
> **Photos** : Pour que le site reste rapide, conseillez-lui d'utiliser des photos de bonne qualité mais pas trop lourdes (idéalement moins de 1Mo).
